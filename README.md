# WeatherAPI-Sarthak

description: A simple weather app using the OpenWeatherMap API

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TheNobody-12/WeatherAPI-Sarthak.git
    ```
2. Install python packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run the app
   ```sh
    python data.py
    ```
